# PSCAN: A Parallel Structural Clustering Algorithm for Big Networks in MapReduce
This repository contains an implementation of PSCAN as authored by Zhao et al. IEEE, 2013. (https://ieeexplore.ieee.org/document/6531844)

## Setup
Instructions for venv setup:

- create venv: `python3 -m venv venv`
- activate venv: `source venv/bin/activate`
- install requirements: `pip install -r requirements.txt`
Note: The venv folder will be ignored by git

## Dataset
Run: `python data/datasets.py` to generate dataset, refer to `data/README.md` for further detail.
Note: generated dataset folders will be ignored by git
